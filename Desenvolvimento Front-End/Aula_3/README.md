Skillet Live Tour 2026

Projeto de página web desenvolvido para divulgação de evento musical da banda Skillet.

Sobre o projeto

Uma Pagina principal criada utilizando HTML e CSS, contendo:

- Navbar personalizada
- Seção hero com imagem de fundo
- Vídeo incorporado do YouTube
- Tabela com cronograma de shows
- Formulário de inscrição
- Player de áudio estilizado

Tecnologias utilizadas

- HTML5
- CSS3

Objetivo

Praticar estruturação de layout, estilização moderna e organização de código.
